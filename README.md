## Project Overview

Vinayak Cyber Zone and Travels is a web application providing services for travel bookings and cyber café facilities.

## Features

- User-friendly interface for travel bookings
- Cyber café service management
- Responsive design
- Real-time availability updates

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Hosting**: Render

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Configure environment variables in `.env`
4. Start the server: `npm start`

## Usage

Navigate to the deployed website to browse and book travel services or access cyber café facilities.
